# Windows Artifact Collector 🔍

**PowerShell-скрипт для сбора цифровых артефактов Windows** на основе YAML-манифеста (формат **Plaso/GRR/Velociraptor**).

Позволяет быстро проверить систему на наличие следов компрометации, изменений конфигураций, механизмов персистентности и других индикаторов, описанных в файле артефактов.

---

## 🚀 Возможности

- ✅ Парсинг YAML-файла с артефактами (стандартный формат от **Plaso/GRR**).
- ✅ Автоматическая подстановка переменных окружения Windows (`%%users.sid%%`, `%%environ_systemroot%%` и др.).
- ✅ Поддержка масок (`*`) в путях реестра и файлов.
- ✅ Сбор:
  - ключей реестра (`REGISTRY_KEY`),
  - значений реестра (`REGISTRY_VALUE`),
  - файлов (`FILE`),
  - выполнение команд (`COMMAND` – например, `netsh`).
- ✅ Цветной структурированный вывод в консоль с отображением разрешённых путей.
- ✅ Подсчёт обработанных артефактов и количества ошибок/не найденных объектов.

---

## 📦 Требования

- **PowerShell 5.1** или новее (включая PowerShell 7).
- **Права администратора** – для доступа к системным разделам реестра и защищённым папкам.
- **Интернет** – для автоматической установки модуля `powershell-yaml` (если он отсутствует).

---

## ⚙️ Установка и запуск

1. **Скачайте скрипт** `app.ps1` и поместите его в любую папку.
2. **Подготовьте YAML-файл** с артефактами (например, `artifactWIN.yaml`).  
3. **Запустите PowerShell от имени администратора** и выполните:

   ```powershell
   .\app.ps1 -yamlPath .\artifactWIN.yaml
   ```

## 🛠️ Как это работает

1. Скрипт проверяет наличие модуля powershell-yaml и при необходимости устанавливает его.
2. Загружает и парсит YAML-файл.
3. Для каждого артефакта определяет тип источника и вызывает соответствующие функции:
  - REGISTRY_KEY – поиск ключей реестра по маске,
  - REGISTRY_VALUE – поиск значений в ключах (с поддержкой *),
  - FILE – поиск файлов по маске,
  - COMMAND – выполнение команды и вывод результата.
4. Все переменные вида %%...%% заменяются на реальные значения из системы (SID пользователя, пути к папкам и т.д.).
5. Результаты выводятся в консоль с цветовой разметкой.
